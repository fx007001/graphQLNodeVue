# vue-graphql

> A Vue.js project

> graphql 入门项目 包含前（vue）、后端（node）的实现  

### 前端 ：
  vue 技术栈 vue + vue-router + vuex  +  axios + vue-apollo + graphql

### 服务端：
  node + express + express-graphql
  
  
### 启动：

  1. 切换到server目录执行 
  ```js
   // 模块安装
   npm i
   // 服务启动
   npm strat
  ```   
  2. 切换到app目录执行
  ```js
   // 模块安装
   npm i
   // 服务启动
   npm start
  ```

### 目录结构 
   - server      服务端目录 \
   -- apiDataes   数据模拟存放目录 \
   -- files       静态文件目录 \
   -- resolver    node解析目录 \
   -- schema      graphql scheam规则定义目录 \
   -- index.js    启动文件 主要的配置都引用  基本都在这里  
   
### 功能实现 

  - 页面  首页，登录，注册
  
  - 功能实现 增删改查
    - 增    注册
    - 删    
    - 改    修改兴趣
    - 查    页面展示
